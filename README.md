# OI Visualizer

信息学奥赛数据结构可视化教学平台 - 面向信息学竞赛学生的现代化数据结构可视化应用

## 项目概述

这是一个基于 Next.js 14 + TypeScript 的数据结构可视化教学平台，支持代码与可视化的双向实时绑定，帮助信息学竞赛学生更好地理解各种数据结构的原理和操作。

## 技术栈

- **Next.js 14+** (App Router) - React 18+ + TypeScript
- **Tailwind CSS** + **shadcn/ui** - 现代化 UI 组件库
- **Zustand** - 轻量级状态管理
- **D3.js v7** - 数据可视化核心库
- **GSAP** - 高性能动画引擎
- **Monaco Editor** - VS Code 同款编辑器

## 功能特性

### 支持的数据结构
- ✅ 数组（一维、二维）
- ✅ 链表（单向、双向、循环）
- ✅ 栈和队列
- ✅ 哈希表
- ✅ 树（BST、AVL、堆）
- ✅ 图（有向、无向、加权）

### 双向绑定
- **正向**：代码编辑 → 解析执行 → 状态变化 → 可视化更新
- **反向**：可视化操作 → 操作映射 → 代码生成 → 编辑器更新

### 交互功能
- 拖拽节点/元素
- 点击选中/编辑值
- 缩放和平移
- 播放/暂停/单步执行动画
- 时间轴拖动

## 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```
